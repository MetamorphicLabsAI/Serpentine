# Serpentine Game Design Concepts: Unlockable Snake Programs

## Standard Challenges (Food Milestones)

1. **Spectrum Shift (Fruit Punch)**
   - **Lore:** A multi-colored shifting anomaly unlocked by eating 20 food in a single standard run. It tastes like a pixelated summer.
   - **Color Pattern:** `Rainbow Shifting`

2. **Crimson Fang**
   - **Lore:** A hostile script designed for rapid data extraction. Unlocked at 30 standard food.
   - **Color Pattern:** `Deep Red / Black Accent`

3. **Cobalt Grid**
   - **Lore:** Cold storage guardian AI. Unlocked at 40 standard food.
   - **Color Pattern:** `Cobalt Blue / Silver`

4. **Emerald Core**
   - **Lore:** Highly optimized organic logic algorithm. Unlocked at 50 standard food.
   - **Color Pattern:** `Bright Green / Lime Glimmer`

5. **Void Star**
   - **Lore:** Born from null pointers and memory leaks. Consumes everything. Unlocked at 60 standard food.
   - **Color Pattern:** `Pitch Black / Violet Strobe`

6. **Plasma Hydra**
   - **Lore:** A multi-threaded monstrosity. Unlocked at 70 standard food.
   - **Color Pattern:** `Hot Orange / Purple`

7. **Monochrome Matrix**
   - **Lore:** Classic retro computing nostalgic snake. Unlocked at 80 standard food.
   - **Color Pattern:** `White / Gray / Black`

8. **Molten Core**
   - **Lore:** Processor overheating manifesting as digital lava. Unlocked at 90 standard food.
   - **Color Pattern:** `Yellow-Orange / Red Hot`

9. **Titanium Coil**
   - **Lore:** Industrial-grade security serpent. Unlocked at 100+ standard food.
   - **Color Pattern:** `Chrome / Steely Blue`

## Speedrunner Challenges (Complete under X time limits)

10. **Quicksilver**
    - **Lore:** A data packet that ignores packet loss. Built purely for speed.
    - **Color Pattern:** `Liquid Silver / Cyan`

11. **Photon Dash**
    - **Lore:** Fiber-optic speedster. The only thing faster is light itself.
    - **Color Pattern:** `Blinding White / Yellow`

12. **Mach Mach**
    - **Lore:** Overclocked baseline architecture. Known to break sound barriers.
    - **Color Pattern:** `Red / Blur`

13. **Sonic Boom**
    - **Lore:** The resultant shockwave of a corrupted CPU clock.
    - **Color Pattern:** `Shocking Blue / White`

14. **Quantum Leap**
    - **Lore:** A protocol that exists in multiple places at once.
    - **Color Pattern:** `Magenta / Cyan Pulse`

15. **Warp Python**
    - **Lore:** A relic of early hyperspace node engineering.
    - **Color Pattern:** `Deep Purple / Starlight White`

16. **Flash Protocol**
    - **Lore:** Injects, executes, and deletes before the system even registers it.
    - **Color Pattern:** `Bright Yellow / Red Accent`
