## 1. UNIVERSAL CONTROLS

These controls work across all modes and menus unless otherwise specified.

| Key | Action |
|-----|--------|
| `Esc` | Pause / Return to previous screen |
| `Enter` / `Space` | Confirm selection |
| `Backspace` | Go back / Cancel |
| `Tab` | Quick menu access |
| `M` | Toggle music mute |
| `S` | Toggle sound effects mute |
| `?` / `F1` | Open controls reference |

### Mercy System (Survival Modes)

| Key | Action |
|-----|--------|
| `Shift` (hold) | Activate Mercy Shield - immune to damage while held |
| Mercy charges | Recharge by collecting Mercy Orbs or surviving 30 seconds |

## 2. STANDARD MODE

Classic snake gameplay - collect data fragments, grow longer, avoid walls and yourself.

### Primary Movement

| Key | Action |
|-----|--------|
| `Arrow Up` / `W` | Move Up |
| `Arrow Down` / `S` | Move Down |
| `Arrow Left` / `A` | Move Left |
| `Arrow Right` / `D` | Move Right |

### Advanced Actions

| Key | Action |
|-----|--------|
| `Space` | Boost (temporary speed increase) |
| `Shift` | Mercy Shield (damage immunity while held) |
| `R` | Restart round (after death only) |
| `Esc` | Pause game |

### Direction Lock

Once you begin moving in a direction, the snake will continue in that direction until you press a new direction key. The snake cannot reverse directly into itself.

## 3. CHRONOSHIFT MODE

Time manipulation survival - master temporal abilities to outlast the corruption.

### Movement Controls

| Key | Action |
|-----|--------|
| `Arrow Up` / `W` | Move Up |
| `Arrow Down` / `S` | Move Down |
| `Arrow Left` / `A` | Move Left |
| `Arrow Right` / `D` | Move Right |

### Time Manipulation Abilities

| Key | Action | Description |
|-----|--------|-------------|
| `Q` | **REWIND** | Rewind time by 3 seconds. Costs 50% Chrono-Meter charge. Cooldown: 15s |
| `W` | **SLOW-MO** | Slow time to 25% speed for 5 seconds. Costs 25% Chrono-Meter charge. Cooldown: 8s |
| `E` | **FAST-FORWARD** | Accelerate time to 200% speed for 4 seconds. Grants temporary invincibility. Costs 30% Chrono-Meter charge. Cooldown: 10s |
| `R` | **TEMPORAL PULSE** | Release a shockwave that damages enemies and pushes back the corruption wall. Costs 75% Chrono-Meter charge. Cooldown: 30s |

### Chrono-Meter HUD

| Key | Action |
|-----|--------|
| `Space` | Activate charged ability (when meter is full) |
| `Shift` | Mercy Shield |
| `Tab` | View Chrono-Meter status |

### Tips

- Chrono-Meter charges passively and by collecting time fragments
- Rewind is your panic button - use it to escape bad situations
- Fast-Forward grants brief invincibility at the start of activation
- Temporal Pulse is most effective when surrounded

## 4. SENTINEL BREACH MODE

Wave-defense survival - protect the Nexus from hostile sentinel programs.

### Movement Controls

| Key | Action |
|-----|--------|
| `Arrow Up` / `W` | Move Up |
| `Arrow Down` / `S` | Move Down |
| `Arrow Left` / `A` | Move Left |
| `Arrow Right` / `D` | Move Right |

### Combat Actions

| Key | Action | Description |
|-----|--------|-------------|
| `Space` | **FIRE** | Shoot projectile toward current direction |
| `Shift` | **NEXUS SHIELD** | Activate defensive shield around Nexus (Mercy charges) |
| `Q` | **DECOY** | Deploy decoy to distract sentinels |
| `E` | **EMP BURST** | Disable all sentinels for 3 seconds (costs 20 energy) |

### Sentinel Types & Priority

| Type | Behavior | Recommended Action |
|------|----------|-------------------|
| PROBE | Basic enemy, low damage | Standard fire |
| SWARMER | Fast, spawns in groups | EMP Burst or focused fire |
| HUNTER | Tracks player, high damage | Use Decoy to redirect |
| TITAN | Slow, high HP, destroys Nexus on contact | Coordinate with teammate or use EMP |

### Wave Management

| Key | Action |
|-----|--------|
| `Tab` | View wave info and enemy count |
| `R` | Force next wave (when all enemies defeated, bonus points) |
| `Esc` | Pause and view threat analysis |

## 8. MOBILE TOUCH CONTROLS

Serpentine features full touch support with a dynamic on-screen D-pad and action buttons.

### D-Pad (Directional Pad)

```
    [UP]
[LEFT] [RIGHT]
   [DOWN]
```

| Touch Area | Action |
|-------------|--------|
| Tap Up zone | Move Up |
| Tap Down zone | Move Down |
| Tap Left zone | Move Left |
| Tap Right zone | Move Right |
| Swipe direction | Quick turn to that direction |

### Action Buttons

| Button | Position | Action |
|--------|----------|--------|
| **A** | Bottom right | Fire / Boost / Execute |
| **B** | Bottom right | Mercy Shield / Cancel |
| **X** | Bottom right | Mode-specific ability 1 (Q) |
| **Y** | Bottom right | Mode-specific ability 2 (E) |

### Touch Gestures

| Gesture | Action |
|---------|--------|
| Single tap | Select / Confirm |
| Double tap | Quick action |
| Long press | Hold action (Mercy Shield) |
| Pinch | Zoom in/out (when available) |
| Two-finger swipe down | Pause game |

### Mobile-Specific

| Key | Action |
|-----|--------|
| `Volume Up` | Brightness up |
| `Volume Down` | Brightness down |
| Swipe from top | Quick settings |
| Swipe from bottom | Quick actions menu |
